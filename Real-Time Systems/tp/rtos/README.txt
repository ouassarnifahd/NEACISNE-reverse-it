****************************
*** Contenu de l'archive ***
****************************


>> cooperative :
Sources de TP pour l'analyse de FreeRTOS en mode cooperatif. 
Certains outils et specifités système seront decouverts

>> doc :
Supports de CM et de TP. Documentations techniques de la maquette et du MCU. 
Documents divers

>> FreeRTOS :
Sources du micro-noyau FreeRTOS avec fichiers portables pour architecture 
PIC32MX de Microchip.

>> peripherals :
projets de gestion des modules UART sur cartes de développement PIC32MAXIWEB 
de Olimex et explorer 16 de Microchip. Une librairie pour la gestion du 
module ADC sur PIC32MX de Microchip est également proposée

>> preemptive :
sources de TP pour l'analyse de FreeRTOS en mode preemptif. 
Certains outils systeme et mecanismes de gestion et protection memoire 
seront decouverts.

>> queue :
sources de TP pour l'analyse de certains outils de communication, 
synchronisation et protection de FreeRTOS en mode preemptif :
queue de message, semaphore, sections critiques

>> tempserver-name1-name2 (modifier avec les noms du binôme) :
repertoire vide dans lequel sera cree le projet de fin de trame de TP
